# JavaGames
Self-designed Java mini-games


# What includes?
* DogAge: Enter your dog's year to see what his/her actual age is!
* LotteryGame: Enter a two-digit number to see how much you have won!

  * DogAge.java 
  * LotterGame.java

* Chinese Version
